JS ACCOUNT SYSTEM WITH MYSQL AND DB CREATION, WITH FE OF REACT + TS + PHASER CANVAS

## For mysql configuration please create a new file

Path for `mysqlConfig.js` -> server-js/mysql/

File should contain:
```
module.exports = {
  mysqlPoolLimit: 5000,
  host     : '',
  user     : '',
  password : '',
  database : ''
}
```
